# soundrop

A Rust clone of Soundrop

[Video of original game](https://youtu.be/63FyqOMpORI)

## Controls

- Left click and drag: draw line
- Right click and drag: delete lines

## Features

- [x] draw lines
- [x] ball spawner
- [x] bounce balls off lines
- [x] delete lines
- [ ] create, delete, and move ball spawners
- [ ] THE SOUND
